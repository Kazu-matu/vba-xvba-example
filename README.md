# XVBA-adin

このリポジトリはExcel VBAプロジェクトのソース管理用です。

## フォルダ構成

- src/ : VBAソースコード（Module, Class, など用途別に管理）
- ribbons/ : リボンUI関連ファイル
- logs/ : ログファイル
- vba-files/ : エクスポート/インポート用VBAファイル
- xvba_modules/ : XVBA関連モジュール
- xvba_unit_test/ : ユニットテスト関連
- config.json, xvba.json : 設定ファイル
- Test.xlsm : Excelマクロブック
- XVBA仕様マニュアル.md : 運用・詳細仕様

## 詳細・運用ルール

詳しい運用方法や仕様は「XVBA仕様マニュアル.md」を参照してください。
